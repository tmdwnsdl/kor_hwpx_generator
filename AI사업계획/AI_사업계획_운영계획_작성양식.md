## [별첨] 고영향 인공지능 사전검토 체크리스트

> 근거: 인공지능기본법 제2조 제4호, 인공지능관리규정 제16조

### 가. 고영향 AI 해당 영역 점검

해당 인공지능이 다음 영역에서 활용되는지 확인합니다.

- ☐ 국민의 생명·신체의 안전에 관한 영역
- ☐ 국민의 기본권 보장에 관한 영역
- ☐ 공공의 안전과 관련된 사회기반시설 운영 영역
- ☐ 그 밖에 대통령령으로 정하는 영역
- ☐ 해당 없음

### 나. 자율성 및 영향도 판단

- ☐ AI의 결과가 사람의 검토 없이 직접 실행되는가?
- ☐ AI 오류 시 인명피해 또는 재산피해 가능성이 있는가?
- ☐ AI 결과가 국민에게 직접 제공되는가?
- ☐ AI가 개인정보 또는 민감정보를 처리하는가?

### 다. 검토 결과

| 항목 | 내용 |
|------|------|
| 검토결과 | ☐ 고영향 해당  ☐ 고영향 비해당  ☐ 판단불가 |
| 판단근거 | |
| 과기정통부 확인요청 | ☐ 필요  ☐ 불필요 |

※ 고영향 해당 시 후속 조치: 총괄부서 통보(제16조④) → 사업자 책무 이행(제17조) → 데이터 품질확인(제18조) → 영향평가(제19조)
